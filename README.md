# sovereign-individual-cn
sovereign-individual 中文翻译项目

希望实现的功能是：
1. 中英对照
2. 良好的排版，能直接打印出来
3. 减少错误，增加注释内容

时间上，看2023年能做完这项工作不。**注意，本项目仅供学习交流使用，未获得原著版权，请勿商用。**

---

2023.09.18
突然发现这个项目没有进行下去的必要了，因为AI工具链已经可以在5分钟内将一本书翻译完，同样是中英对照。用 z-library 下载本书的 epub 版，然后用 [Immersive translate](https://immersivetranslate.com/) 插件将书籍进行整体翻译。两个步骤，五分钟搞定。

根据上面的操作流程，我已经弄好《主权个人》的中文版。
接下来，就好好读，暂时不折腾格式和排版问题了。
